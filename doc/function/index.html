    Load libR from appropriate place.

    Params:

    NameTypeAttributeDescription
    r_path *

    "auto" (default) || "environment" || "system" || "buildin" || path to R_HOME "auto" - try "environment" -> "system" -> "buildin" "environment" - use environmental R_HOME and LD_LIBRARY_PATH. libr-bridge handles nothing. "system" - find system installed R "buildin" - (NOT IMPLEMENTED) use redistributed binary attached with libr-bridge-bin
